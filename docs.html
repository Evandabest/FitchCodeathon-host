API Documentation

Sign Up

Endpoint: /EcoCycle/signUp/<firstName>/<lastName>/<userName>/<email>/<password>/<referallCode>

Method: GET

Sample Response:

If there is no referallCode than pass in a string called 'None'

{
  "message": "Sign up successful"
}

Log In

Endpoint: /EcoCycle/logIn/<email>/<password>

Method: GET

Sample Response:

{
  "message": "Login successful"
}

Log In Vendor

Endpoint: /EcoCycle/logInVendor/<username>/<password>

Method: GET

Sample Response:

{
"message": "Login successful"
}
