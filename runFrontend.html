How to Run the Frontend

Follow these steps to run the frontend application:

  1. Ensure you have Node.js installed on your machine.
  2. Open a terminal and navigate to the frontend directory: 
    3. cd frontend
  3. Install the dependencies by running:
    4. npm install
  4. Start the development server:
    5. npm run dev
  5. Open your browser and navigate to http://localhost:3000 to see the application running.
  6. Haven't set up you're backend? Than go to this page to set it up: Run Backend
  7. Please note, our backend is on ANOTHER branch , so on a different terminal open up the project and type in "git checkout flaskComp" to get into the the branch or try another way of getting into that branch
